- Titolo progetto -
 port_scanner – Python TCP Port Scanner

- Descrizione -
Ho sviluppato uno scanner di porte TCP in Python, eseguibile da linea di comando. Supporta scansioni multithreaded su intervalli di porte personalizzati e salva i risultati in un file JSON. L'obiettivo era creare uno strumento leggero, semplice ma professionale, da pubblicare su GitHub come prova concreta di competenze tecniche di base in rete e scripting.

- Tecnologie usate -
Python 3
argparse
socket
threading
json
Git + GitHub
Git Bash / PowerShell

- Struttura del progetto - 
scanner.py: script principale
open_ports.json: file output JSON
README.md: documentazione progetto
requirements.txt: dipendenze (vuoto)
LICENSE: licenza MIT

- Funzionalità -
Scansione TCP su IP e range di porte specificato
Output CLI + salvataggio su file
Multithread per aumentare la velocità
Parametri da linea di comando con argparse
Gestione errori base
Nessuna dipendenza esterna
